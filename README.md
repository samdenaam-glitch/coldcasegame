A minimalistic web detective game where players solve cold cases from a virtual police archive.

## Features
- Solve mysterious cold cases
- Analyze evidence and interview witnesses
- Minimalistic dark theme interface
- Progressive difficulty system
- Save your detective progress

## How to Play
1. Open `index.html` in your browser
2. Select a cold case from the archive
3. Examine evidence and interview witnesses
4. Take notes in your detective's notepad
5. Submit your solution when ready

## Development Setup
Simply open `index.html` in any modern web browser.

## Technologies Used
- HTML5, CSS3, JavaScript
- LocalStorage for game saves
- JSON for case data
